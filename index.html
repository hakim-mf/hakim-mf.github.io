GIM (Gravity Interactive Modelling)

GravityForward ModellingSubsurface Analysis3D Geometry

Interactive modeling software for synthetic gravity anomalies. Designed as an educational instrument with an intuitive Graphical User Interface (GUI) to bridge the gap between theoretical formulas and practical interpretation.

View Specifications & Download

Key Features:

  • Real-time Visualization: Instant spatial visualization of gravity anomaly (gz) variations on the Map Canvas.
  • Multi-Body Modeling: Create complex subsurface scenarios by combining multiple geometric bodies, including Spheres, Rectangular Prisms, and Fault models.
  • Cross-Section Analysis: Integrated slicing tool (A-A') to directly compare gravity profile curves against subsurface model sections.
  • Data Management: Save and reload projects in JSON format, export results to CSV for external analysis, and capture high-quality map images.

Computational Methodology: The calculation engine utilizes high-precision analytical solutions. For spherical geometries, it implements the Telford et al. (1990) formula. For rectangular prisms, it employs the Plouff (1976) closed-form solution in Arctan form to ensure superior numerical stability and avoid singularities.
